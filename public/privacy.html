QC Audit Tracker – Privacy Policy

Last Updated: [replace with date]

QC Audit Tracker is a Chrome extension that helps Outlier AI auditors track time spent on audits and view analytics and exports. The extension operates only on https://app.outlier.ai/*. All audit data stays on your device unless you explicitly export it.

Data We Collect

  • Audit metadata (from Outlier pages you visit): Project ID and project name, QA operation ID, attempt ID, review level; audit timing (start/end, completion/transition timestamps, durations); per‑project max time (derived from page APIs or a remote configuration map).
  • Off‑platform time entries (user‑provided): Activity type, description, date, and duration entered by you.
  • Settings and overrides (user‑provided): Project name overrides and max time overrides; optional email for timesheet export; preference toggles (e.g., developer logging).
  • Configuration fetched remotely: A small read‑only JSON map of projectId → maxTime (minutes) from a Google Apps Script URL. This is configuration data only; it contains no personal data or executable code.
  • Diagnostics: Optional in‑extension logs for troubleshooting. These appear in your browser console only and are not transmitted.

How We Collect Data

Content and page scripts read relevant audit data from specific Outlier API responses and the page UI while you are on app.outlier.ai. The extension fetches the project max‑time map once per day (or on startup) from the configured URL. You may enter off‑platform entries and overrides via the extension UI.

How We Use Data

  • Provide accurate time tracking, analytics, and exports (e.g., timesheets).
  • Apply correct per‑project max time to timers and analytics.
  • Maintain your settings and overrides across sessions.
  • Perform background health checks and daily refresh of the max‑time map.

Data Sharing

  • We do not sell or share your data with third parties.
  • No analytics, ad networks, or tracking services are used.
  • The only network request is to fetch the configuration JSON (project max times). No user data is sent in this request.
  • Exports (e.g., CSV/TSV) are initiated by you. If you share exported files, that sharing is under your control.

Data Storage & Retention

Data is stored locally using Chrome’s storage.local on your device (tasks, timers, off‑platform entries, overrides, settings). Data persists until you delete it from the extension, clear your browser data, or uninstall the extension.

Security

  • Data remains on your device by default and is not transmitted to our servers.
  • Host permissions and code are limited to Outlier pages to minimize access surface.
  • The remote configuration fetch is read‑only JSON, parsed as data (no remote code is executed).

Your Choices & Controls

  • Enable/disable tracking from the extension settings.
  • Edit or delete off‑platform entries and project overrides.
  • Clear stored data by removing entries or by clearing site data for the extension in Chrome.
  • Uninstalling the extension removes all stored data from your device.

Children’s Privacy

This extension is intended for adult professional users. It is not directed to children under 13.

Regional Rights (GDPR/CCPA)

We do not run servers or store personal data outside your device. If you have questions about local data stored by the extension or wish to exercise data rights, you can remove or modify the data directly in the UI or uninstall the extension. Permissions Disclosure

  • storage: store timers, audit history, settings, overrides, and the project max‑time map locally.
  • alarms: keep timers accurate, perform hourly health checks, and refresh the max‑time map daily.
  • tabs: send timer updates to Outlier tabs (no browsing history collection).
  • Host permissions:
    • https://app.outlier.ai/* to operate only on Outlier pages.
    • https://script.google.com/* and https://script.googleusercontent.com/* to fetch the project max‑time configuration JSON.

Remote Code

No. The extension does not execute remote code. The only remote call fetches a static JSON configuration, which is parsed as data.
